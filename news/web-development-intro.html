HTML Introduction

(ክረድት: @Henok_Henzo)
HTML ኤችቲኤምኤል ምፃረ ቃሉ የቆመው ለHyper Text Markup Language ሲሆን ዌብሳይቶችን በመስራት ውስጥ ከፍተኛ ድርሻ ያለው ቋንቋ ነው::

Hypertext የሚለው ቃል የሚያመለክተው እንዴት የተለያዩ የዌብሳይት ገፆች(HTML ዶክመንቶች) እርስ በእርስ እንደሚገናኙ ወይም Linked እንደሆኑ ነው::
ስለዚህም በድህረ ገፆች ላይ የሚገኙ ሊንኮች(Links) ሀይፐርቴክስት(hypertext) ተብለው ይጠራሉ::

Markup Language የሚለው በስሙ ውስጥ የሚገኘው ቃል አንድን ነገር ምልክት ማድረግን የሚያመለክት ሲሆን
የHTML ፋይል ደግሞ የተለያዩ መለያዎችን(Tags) በመጠቀም ለዌብ ብራውሰራችን እዴት አድርጎ የፃፍነውን ነገር እንዲያሳይ የምናዝበትን አቅም ሚሰጠን ቋንቋ ነው::

በመጀመርያ አከባቢ HTML የተሰራው ሳይንሳዊ የሆኑ መረጃዎችን በሪሰርቸሮች መሀከል በቀላሉ እንዲገናኙ ለማድረግ ታስቦ የተሰራ ሲሆን ሼር የሚደራረጉትን ፅሁፍ አወቃቀር ለማደራጀት ታስቦ የተሰራ ነበር::
በዛን ጊዜ ይህ የHTML አወቃቀር በውስጡ የአርእስት, አንቀፅ, ዝርዝሮችን የመሳሰሉ መለዮዎችን የያዘ ነበር::

በአሁን ሰአት HTML በተለያዩ ዌብሳይቶች ላይ በስፋት የሚገኝ ሲሆን የተለያዩ ብዛት ያላቸው መለዮዎችም(Tags) ለተለያዩ ጥቅሞች ተጨምሮባቸዋል::
አሁን ላይ HTML version 5 ላይ ስለሆነ በዚህ ፅሁፍ ምንመለከታቸው ነገሮች በአዲሱ ስሪት ላይ ያተኮረ ይሆናል::
በምስሉ ላይ የምትመለከቱት ስእል ላይ ከላይ የሚታየው መሰረታዊ የሆነ የHTML ዶክመንት( ወደፊት በዝርዝሩ ምንመለስበት ይሆናል ) ሲሆን ከታች ያለው ደግሞ ኮዳችንን ስንከፍተው የሚወጣው የመጀመርያ ድህረገፃችን ገፅ ነው ነው::

#Web Development For Absolute Beginner


ዛሬ በዝች ፅሁፍ የመጀመርያ ዌብሳይታችንን እንሰራለን።

ይህን ዌብሳይት ለመስራት ከሚያስፈልጉን ነገሮች ውስጥ አንደኛው ኮድ ኤዲተር(Code editor) ነው ይህንንም የኢንተርኔት አገልግሎት በኮምፒውተር ላይ ከሌላቹ ከራሱ ኦፕሬቲንግ ሲስተም ላይ ተጭኖ የመጣ ሶፍትዌር ስላለ እሱን እንጠቀማለን።
(ለወደፊት ግን ሌላ ሶፍትዌር ዳውንሎድ አድርገን እንጭናለን)

በዚህ ፅሁፍ የምንጠቀመው ኖት ፓድ(Notepad) የሚባለውን የዊንዶውስ ሶፍትዌር ሲሆን እሱንም ለመክፈት ሁለት መንገዶች አሉ።
1- ስታርት ሜኑን ክሊክ በማድረግ ወደ ኦል ፕሮግራምስ በመጫን (All programs) ቀስታችንን ወደ አክሰሰሪስ (Accessories) ስንነካ ኖትፓድ(Notepad) ሚለውን እናገኛለን።

2- ስታርትን በመጫን ሰርች ማድረግያ(Search box) ውስጥ Notepad ብሎ በመፃፍ መጀመርያ ሚመጣውን ሶፍትዌር መክፈት።

ከላይ ከተጠቀሱት መንገዶች አንዱን በመጠቀም ከከፈታችሁ ምንም ፅሁፍ የሌለው ቴክስት ኤዲተር ታገኛላችሁ።
በዛሬው ፅሁፍ ላይ የምንጠቀመው ቋንቋ HTML(HyperTextMarkupLanguage) የሚባለውን ነው።
HTML ራሱን የቻለ የፕሮግራሚንግ ቋንቋ ባይሆንም ሁሉም ዌብሳይት የHTML ኮድ ሊኖረው ግድ ይላል።
ብዙ ጊዜ ዌብሳይት ለመስራት ከሚያስፈልጉ ቋንቋዎች ውስጥ HTML, Css እና javascript የሚባሉት ይጠቀሳሉ እንዚህም መሰረታዊ የዌብሳይት ቋንቋዎች ናቸው።
(ተጨማሪ ትምህርቶች በቅርቡ ይዘን የምንቀርብ ይሆናል።) የHTML ጥቅም አንዱ የምንሰራውን ዌብሳይት ማደራጀት ነው።
HTML በውስጡ html elements የሚባሉ ስብስቦችን አቅፏል እነዚህም የተለያዩ ነገሮችን ከፋፍለን እንድንይዝ ይረዱናል።
የኤችቲኤምኤል ኤለመንት በታግ(tag) ይወከላል ይህም html tag ተብሎ ይጠራል። HTML tag ለመፃፍ በመጀመርያ " < " ምልክትን እንፅፋለን ከዚያም በመቀጠል የታጉን ስም እንፅፋለን በመቀጠልም " > " ምልክት እናስቀምጣለን። ከላይ የፃፍነው የሚከተለውን መልክ ሊኖረው ይገባል።

<tagname>
ይህ ከላይ ያስቀመጥነው opening tag (ኦፕኒንግ ታግ) የሚባል ሲሆን ከስሙ እንደምንረዳው ከላይ የፃፍነው የተከፈተ ታግ እንደሆነ ነው።
ከላይ የከፈትነውን ታግ ለመዝጋት ደግሞ እንደመጀመርያው "<" ምልክት ካደረግን ቡሀላ "/" የሚለውን እናስከትላለን።
በመቀጠልም ቅድም የፃፍነውን ታግ ስም(tagname) እናደርጋለን በመጨረሻም " > " የሚለውን እንፅፋለን ይህም የሚከተለውን ሊመስል ይገባል።
</tagname>
ከላይ የመክፈቻውን ታግ እና የመዝግያውን ታግ በአንድ ላይ ስንፅፈው
<tagname></tagname>

የሚለውን እናገኛለን። በመቀጠል በፃፍነው ታግ መሀከል ላይ እንዲታይ የምንፈልጋቸውን ነገሮች እንደፈለግን እንፅፋለም ይህም እንደሚከተለው ይሆናል።

<tagname>This is a content</tagname>

አሁን ከላይ የተማርነውን መሰረት በማድረግ HTML ላይ ያሉትን ታጎች በመጠቀም ዌብሳይታችንን እንሰራለን።
ማንኛውም ዌብሳይት ከመስራታችን በፊት በመጀመርያው መስመር ላይ <!DOCTYPE html> ብለን እንፅፋለን ይህም ለብሮውሰሩ(browser) የኤችቲኤምኤል ቨርዥን 5(version 5) እንደምንጠቀም ይነግረዋል።
በመቀጥል ከታች <html></html> ብለን እንፅፋለን ይህም root element (ሩት ኤለመንት) ተብሎ ይጠራል።
ይህም ማንኛውም ምንፅፋቸው ኮዶችን አቅፎ ይይዛል። ከዚህ በኋላ ያሉትን ኮዶች ለመረዳት እንዲቀላችሁ በየደረጃው እፅፈዋለው። እስካሁን የፃፍነው ከታች ያለውን ሊመስል ይገባል።

<!DOCTYPE html> <html></html>

ከዚህ በመቀጠል hmtl ታግ ውስጥ opening head tag (በውስጡ head የሚል ስም ያለው ታግ) <head> እንከፍት እና እንዘጋለን </head>።
ከዚያም ከ head tag መዝግያው ታች<body></body>ብለን እንቀጥላለን ይህም ከታች ያለውን ይመስላል።

ምሳሌ

    <!DOCTYPE html>
    <html>
       <head>
         <title></title>
       </head>
      <body>
        
      </body>
    </html>


ከላይ የፃፍነው head tag እና body tag የተለያዩ ወሳኝ ጥቅሞች አሏቸው። የመጀመርያው head tag በዌብሳይቱ ላይ የማይታዩ ነገር ግን ለብሮውሰሩ መረጃ የሚሰጡትን ታጎች አቅፎ ይይዛል ከእነዚህም መሀከል

የሚባለው ተጠቃሽ ነው ይህም የዌብሳይቱን ስም ልንፅፍበት እንችላለን። ከታች ለዚህ ምሳሌ እንፃፍ

ምሳሌ

    <!DOCTYPE html>
    <html>
       <head>
         <title>This is a title</title>
       </head>
      <body>
        <h1> </h1>
        <p>This is a paragraph(አንቀፅ ነው)</p>
      </body>
    </html>

ከላይ አስተውላችሁ ከተመለከታችሁ title ሚለውን ታግ ወደ ውስጥ ስፔስ አድርጌ ፅፌዋለው ይህም ለማንበብ እንዲረዳኝ ነው እናንተም እንዲህ ካደረጋችሁ ወደፊት ብዙ ኮድ ስትፅፉ አይቸግራችሁም።
