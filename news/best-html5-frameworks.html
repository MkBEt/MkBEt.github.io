Best HTML5 Frameworks

በአሁን ሰዓት የምንሰራቸውን ድህረገፆች ይበልጥ የተዎቡ እና በተለያዩ browser የተስተካከለ እይታ እንድኖረው (responsive) ለማድረግ የተለያዩ Framework መጠቀም እየተለመደ መቷል። FrameWork የምባሉት ቀጥኛ ፍችያቸው ስናይ
"A framework is defined as a package made up of a structure of files and folders of standardized code (HTML, CSS, JS documents etc.)
which can be used to support the development of websites, as a basis to start building a site."

ለምሳሌ Bootstrap አንድ የደህረ ገፅ አናት ላይ የምቀመጠው Navgationbar በCSS ለመስራት ከምንፅፈው እጅግ ብዙ ኮዶች የHTML class ላይ nav navbar የምል ስም ብቻ በመጨመር መሠረታው ነገሮችን እራሱ ይጨምርልናል። የበለጠ ያንብቡ


1, Twitter Bootstrap

Bootstrap የHTML, CSS Framework ስሆን ይበልጥ ድህረ-ገፃችን በተለያዩ browser የተስተካከለ እይታ እንድኖረው የምረዳ ነው።ለመጀመርያ ግዜ በtwitter የተሠራ ስሆን ለመጠቀም ስራ በመጀመሪያ bootstrap ፋይል ዳውንሎድ ማድረግ ይኖሪብናል፣ በአሁን ሰዐት ጥቅም ላይ እየዋለ የለው bootsrap4 ከዚህ ሊንክ ያውርዱ ⬇

2, Foundation

በZURB ድርጅት ባለቤትነት የተሠራው ይህ framework እጅግ ዘመናው እና ሙሉ ለሙሉ ነፃ ነው። ይህን ለየት የምያደርገው በአንድ ኮድ ከትልቅ ስክርን እስከ ሞባይል የተስተካከለ ድህረ ገፅ ለመስራት ተመራጭ ነው።

3, Skeleton



ሌላኛው ታዎቅው Framework ስሆን ቀላል እና ንፁህ CSS boilerplate ነው። ለድህረ ገፅ እንድሁም ለሞባይል አፕ ስራ ይጠቅማል ይሞክሩት።

Some notable features: a responsive layout grid, standard media queries for your device-specific CSS style properties, a CSS class for responsive image elements that scale with the layout grid, a PSD template for mocking up your web designs, and an HTML5 shiv for old web browsers.
If you want to get started with responsive design quickly, you should look into this open source project.

4, HTML5 Boilerplate



እኤአ በ 2010, HTML5 Boilerplate የመጀመሪየው የወብሳይቶች የፊት ገፅ (front end) ማስዋብያ Software ሆኗል።

The most popular open source front-end web development tools for getting HTML5 websites and web apps up and running in no time. It’s a compilation web development solutions that enable our sites to support modern web browsers.
