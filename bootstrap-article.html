Bootstrap Highlight

Bootstrap የHTML, CSS Framework ስሆን ይበልጥ ድህረ-ገፃችን በተለያዩ browser የተስተካከለ እይታ እንድኖረው የምረዳ ነው።ለመጀመርያ ግዜ በtwitter የተሠራ ስሆን ለዛሬህ ስራ በመጀመሪያ bootstrap ፋይል ዳውንሎድ ማድረግ ይኖሪብናል፣ በአሁን ሰዐት ጥቅም ላይ እየዋለ የለው bootsrap4 ከዚህ ሊንክ ያውርዱ > bootstrap4.zip Download ትዕዛዞቹን ይከተሉ

Steps

First download bootstrap software then create new folder (eg. "assets")
put the file to the new folder
Go to your html file and import bootstrap by writing the correct file path. Example add this code in head tag. 
<link rel="stylesheet" type="text/css" href="assets/bootstrap.min.css">
